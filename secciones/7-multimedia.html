Etiquetas multimedia

Estas etiquetas se utilizan para insertar y controlar contenido multimedia en la página web, como imágenes, audio y video.

<img>

Se utiliza para insertar imágenes en la página web. Se coloca la ruta o URL como valor del atributo src y un texto alternativo como valor del atributo alt.

Se puede modificar la altura y anchura con los atributos width y height. Se puede usar el atributo loading="lazy" para que la imagen se descargue cuando estemos cerca del elemento, y el atributo decoding="async" para procesar sin bloquear el renderizado. También se puede usar el atributo fetchpriority="high" para indicar al navegador que priorice la descarga de imágenes importantes como la imagen principal (LCP).

info

Nota

Establecer dimensiones evita cambios de layout inesperados mientras las imágenes cargan.

HTML 
<img
  src="ruta/de/la/imagen.jpg"
  alt="Texto alternativo"
  width="200"
  height="150"
/>
lightbulb

Tip

Mantén las imágenes lo más ligeras posible, idealmente por debajo de 100 KB. Las imágenes pesadas ralentizan la carga de la página y afectan la experiencia del usuario.

link

Recursos útiles

  • Reducir el tamaño de imágenes: TinyPNG
  • Quitar metadatos de imágenes: VerExif
  • Descargar imágenes gratis: Pexels

<picture> </picture>

Permite definir múltiples fuentes de imagen para que el navegador elija la más adecuada según el tamaño de pantalla o el formato soportado. Se utiliza junto con la etiqueta <source> para definir las condiciones, y siempre debe incluir un <img> como respaldo, por ejemplo:

HTML 
<picture>
  <source srcset="imagen.avif" type="image/avif" />
  <source srcset="imagen.webp" type="image/webp" />
  <source srcset="imagen-grande.jpg" media="(min-width: 800px)" />
  <img src="imagen.jpg" alt="Descripción de la imagen" />
</picture>

<audio> </audio>

Se utiliza para insertar contenido de audio en la página web. Usa el atributo controls para agregar los controles al audio. Usa la etiqueta <source> dentro del contenido para colocar distintos formatos, por ejemplo:

HTML 
<audio controls>
  <source src="ruta/del/audio.mp3" type="audio/mpeg" />
  <source src="ruta/del/audio.ogg" type="audio/ogg" />
  Tu navegador no soporta el elemento de audio.
</audio>

<video> </video>

Se utiliza para insertar contenido de video en la página web. Acepta los siguientes atributos:

  • controls: agrega los botones de control del video.
  • autoplay: el video se reproduce automáticamente.
  • loop: el video comienza a reproducirse de nuevo cada vez que termina.
  • muted: el contenido se reproduce con el sonido desactivado de forma predeterminada.
  • poster: la URL de una imagen que se mostrará antes de que se reproduzca el video.
  • preload: indica al navegador que cargue el video de antemano.

Usa la etiqueta <source> dentro del contenido para colocar distintos formatos, por ejemplo:

HTML 
<video controls autoplay loop muted preload="auto" poster="poster.png">
  <source src="ruta/del/video.mp4" type="video/mp4" />
  <source src="ruta/del/video.ogg" type="video/ogg" />
  Tu navegador no soporta el elemento de video.
</video>

<track>

Se utiliza dentro de <video> o <audio> para agregar subtítulos, descripciones o capítulos al contenido multimedia. Mejora significativamente la accesibilidad. Acepta los siguientes atributos:

  • src: ruta al archivo de subtítulos (formato .vtt).
  • kind: tipo de pista (subtitles, captions, descriptions).
  • srclang: idioma de la pista (ej. es, en).
  • label: nombre visible para el usuario.
  • default: activa la pista por defecto.
HTML 
<video controls>
  <source src="video.mp4" type="video/mp4" />
  <track
    src="subtitulos-es.vtt"
    kind="subtitles"
    srclang="es"
    label="Español"
    default
  />
  <track
    src="subtitulos-en.vtt"
    kind="subtitles"
    srclang="en"
    label="English"
  />
</video>

<iframe> </iframe>

Permite incrustar contenido externo dentro de la página web, como videos de YouTube, mapas de Google o cualquier otro sitio web. Acepta los siguientes atributos:

  • src: URL del contenido a incrustar.
  • width y height: dimensiones del iframe.
  • title: descripción del contenido para accesibilidad.
  • loading="lazy": carga el iframe solo cuando sea visible.
  • allow: permisos que se otorgan al contenido incrustado.
  • allowfullscreen: permite que el contenido se muestre en pantalla completa.
HTML 
<iframe
  src="https://www.youtube.com/embed/video-id"
  width="560"
  height="315"
  title="Título del video"
  loading="lazy"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media"
  allowfullscreen
></iframe>
warning

Seguridad

Solo incrusta contenido de fuentes confiables. El atributo sandbox permite restringir las acciones del contenido incrustado para mayor seguridad.
