Etiquetas para menús

Estas etiquetas permiten crear elementos de selección interactivos para que los usuarios puedan elegir entre diferentes opciones.

<datalist> </datalist>

Se utiliza para proporcionar una lista de opciones predefinidas para un campo de entrada. El usuario puede escribir libremente o seleccionar de las opciones sugeridas.

Usa la etiqueta <input> con el atributo list para asociar el campo de entrada con la lista de opciones. El valor del atributo list debe coincidir con el id del <datalist>.

Usa la etiqueta <option> dentro de <datalist> para definir las opciones disponibles, por ejemplo:

HTML 
<input list="navegadores" name="navegador" />
<datalist id="navegadores">
  <option value="Chrome"></option>
  <option value="Firefox"></option>
  <option value="Safari"></option>
  <option value="Edge"></option>
  <option value="Opera"></option>
</datalist>
Resultado

<select> </select>

Se utiliza para crear un menú desplegable donde el usuario debe seleccionar una opción de la lista.

Usa la etiqueta <option> dentro de <select> para definir las opciones disponibles. El atributo value define el valor que se enviará al servidor, mientras que el contenido de texto es lo que verá el usuario, por ejemplo:

HTML 
<select name="frutas">
  <option value="">--Selecciona una fruta--</option>
  <option value="manzana">Manzana</option>
  <option value="banana">Banana</option>
  <option value="naranja">Naranja</option>
  <option value="pera">Pera</option>
</select>
Resultado

<optgroup> </optgroup>

Se utiliza para agrupar opciones relacionadas dentro de un <select>. Esto ayuda a organizar listas largas de opciones y mejora la experiencia del usuario. El atributo label define el nombre del grupo que se mostrará al usuario, por ejemplo:

HTML 
<select name="vehiculo">
  <option value="">--Selecciona un vehículo--</option>
  <optgroup label="Autos">
    <option value="sedan">Sedán</option>
    <option value="suv">SUV</option>
    <option value="coupe">Coupé</option>
  </optgroup>
  <optgroup label="Motocicletas">
    <option value="deportiva">Deportiva</option>
    <option value="cruiser">Cruiser</option>
    <option value="touring">Touring</option>
  </optgroup>
</select>
Resultado

Atributos importantes

Los elementos de menú soportan varios atributos que modifican su comportamiento:

selected

Marca una opción como seleccionada por defecto cuando la página carga. Solo debe haber una opción con este atributo en un <select> simple, por ejemplo:

HTML 
<select name="pais">
  <option value="mx" selected>México</option>
  <option value="es">España</option>
  <option value="ar">Argentina</option>
</select>
Resultado

disabled

Deshabilita una opción específica para que no pueda ser seleccionada, o deshabilita el <select> completo. Es útil para opciones que no están disponibles temporalmente, por ejemplo:

HTML 
<select name="talla">
  <option value="">--Selecciona una talla--</option>
  <option value="s">S - Pequeña</option>
  <option value="m" disabled>M - Mediana (Agotada)</option>
  <option value="l">L - Grande</option>
  <option value="xl">XL - Extra grande</option>
</select>
Resultado

multiple

Permite al usuario seleccionar múltiples opciones. El usuario puede mantener presionada la tecla Ctrl (Windows/Linux) o Cmd (Mac) para seleccionar varias opciones, por ejemplo:

HTML 
<select name="idiomas" multiple>
  <option value="es">Español</option>
  <option value="en">Inglés</option>
  <option value="fr">Francés</option>
  <option value="de">Alemán</option>
  <option value="it">Italiano</option>
</select>
Resultado

size

Define cuántas opciones visibles se mostrarán a la vez. Por defecto, un <select> muestra solo una opción y se despliega al hacer clic. Con size, puedes mostrar múltiples opciones simultáneamente, por ejemplo:

HTML 
<select name="color" size="4">
  <option value="rojo">Rojo</option>
  <option value="azul">Azul</option>
  <option value="verde">Verde</option>
  <option value="amarillo">Amarillo</option>
  <option value="morado">Morado</option>
  <option value="naranja">Naranja</option>
</select>
Resultado

required

Hace que el campo sea obligatorio. El formulario no se enviará hasta que el usuario seleccione una opción válida (diferente de la vacía), por ejemplo:

HTML 
<form>
  <label for="categoria">Categoría:</label>
  <select name="categoria" id="categoria" required>
    <option value="">--Selecciona una categoría--</option>
    <option value="tech">Tecnología</option>
    <option value="design">Diseño</option>
    <option value="marketing">Marketing</option>
  </select>
  <button type="submit">Enviar</button>
</form>
Resultado
lightbulb

Tip

La diferencia principal es que <datalist> permite al usuario escribir libremente y solo sugiere opciones, mientras que <select> limita al usuario a seleccionar solo de las opciones disponibles.
