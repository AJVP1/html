Etiquetas de tipo lista

Hay tres tipos de listas: las ordenadas, las desordenadas y las listas de descripción.

<ul> </ul>

Para crear listas desordenadas, el navegador nos muestra los elementos de la lista con viñetas. Por ejemplo:

HTML 
<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
  <li>Elemento 3</li>
</ul>

Resultado

  • Elemento 1
  • Elemento 2
  • Elemento 3

<ol> </ol>

Para crear listas ordenadas, el navegador nos muestra los elementos de la lista con números. Por ejemplo:

HTML 
<ol>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
  <li>Elemento 3</li>
</ol>

Resultado

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Podemos usar el atributo start para especificar de qué número empezar el conteo, por ejemplo:

HTML 
<ol start="5">
  <li>Elemento 1</li>
  <li>Elemento 2</li>
  <li>Elemento 3</li>
</ol>

Resultado

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Podemos usar el atributo type para cambiar el tipo de numeración. Los valores posibles son 1 (números), a (letras minúsculas), A (letras mayúsculas), i (números romanos en minúscula) e I (números romanos en mayúscula), por ejemplo:

HTML 
<ol type="a">
  <li>Elemento 1</li>
  <li>Elemento 2</li>
  <li>Elemento 3</li>
</ol>

Resultado

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

También podemos usar el atributo reversed para invertir el orden del conteo, por ejemplo:

HTML 
<ol reversed>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
  <li>Elemento 3</li>
</ol>

Resultado

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

<dl> </dl>

Para crear listas de descripción, el navegador nos muestra los términos y sus descripciones. Usa <dt> para definir el término y <dd> para definir la descripción del término. Por ejemplo:

HTML 
<dl>
  <dt>Término 1</dt>
    <dd>Descripción del término 1</dd>
  <dt>Término 2</dt>
    <dd>Descripción del término 2</dd>
</dl>

Resultado

Término 1
Descripción del término 1
Término 2
Descripción del término 2

<li> </li>

Define cada elemento dentro de una lista ordenada (<ol>) o desordenada (<ul>). Dentro de una lista ordenada, podemos usar el atributo value para asignar un número específico a un elemento, y los siguientes continuarán a partir de ese valor, por ejemplo:

HTML 
<ol>
  <li>Elemento 1</li>
  <li value="10">Elemento 2</li>
  <li>Elemento 3</li>
</ol>

Resultado

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
lightbulb

Tip

Las listas se pueden anidar unas dentro de otras para crear estructuras más complejas. Por ejemplo:

HTML 
<ul>
  <li>Frutas
    <ol>
      <li>Manzana</li>
      <li>Naranja</li>
    </ol>
  </li>
  <li>Verduras
    <ol>
      <li>Zanahoria</li>
      <li>Brócoli</li>
    </ol>
  </li>
</ul>

Resultado

  • Frutas
    1. Manzana
    2. Naranja
  • Verduras
    1. Zanahoria
    2. Brócoli
