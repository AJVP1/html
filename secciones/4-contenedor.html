Etiquetas de tipo contenedor

Estas etiquetas se utilizan para estructurar y organizar el contenido de la página web, proporcionando una semántica clara sobre la función de cada sección.

<header> </header>

Define el encabezado de la página web, generalmente contiene el logotipo, el título y la barra de navegación, por ejemplo:

HTML 
<header>
  <h1>Mi Sitio Web</h1>
  <nav>...</nav>
</header>

<nav> </nav>

Define la barra de navegación de la página web, generalmente contiene enlaces a las diferentes secciones del sitio web. Se recomienda usar el atributo aria-label para describir el propósito de la navegación, especialmente cuando hay múltiples elementos <nav> en la misma página, por ejemplo:

HTML 
<nav aria-label="Navegación principal">
  <a href="/inicio">Inicio</a>
  <a href="/acerca">Acerca de</a>
  <a href="/contacto">Contacto</a>
</nav>

<main> </main>

Define el contenido principal de la página web, es decir, la parte más importante del sitio, por ejemplo:

HTML 
<main>
  <h1>Bienvenido a mi sitio</h1>
  <p>Este es el contenido principal.</p>
</main>

<aside> </aside>

Define contenido adicional relacionado con el contenido principal, como barras laterales, citas o enlaces relacionados, por ejemplo:

HTML 
<aside>
  <h2>Artículos relacionados</h2>
  <ul>
    <li><a href="#">Artículo 1</a></li>
    <li><a href="#">Artículo 2</a></li>
  </ul>
</aside>

<footer> </footer>

Define el pie de página de la página web, generalmente contiene información de contacto, derechos de autor y enlaces a redes sociales, por ejemplo:

HTML 
<footer>
  <p>&copy; 2025 Mi Sitio Web</p>
  <a href="/contacto">Contacto</a>
</footer>

<section> </section>

Define una sección temática de contenido dentro de la página web. Cada <section> debería incluir un encabezado ( <h2> , <h3> , etc.) que identifique el tema de esa sección, por ejemplo:

HTML 
<section>
  <h2>Sobre nosotros</h2>
  <p>Somos una empresa dedicada a...</p>
</section>

<section>
  <h2>Nuestros servicios</h2>
  <p>Ofrecemos los siguientes servicios...</p>
</section>

<article> </article>

Define un artículo independiente dentro de la página web, como una entrada de blog o una noticia, por ejemplo:

HTML 
<article>
  <h2>Título del artículo</h2>
  <p>Contenido del artículo...</p>
  <p>Publicado el 12 de febrero de 2025</p>
</article>

<figure> </figure>

Define un contenedor para contenido multimedia, como imágenes, gráficos o videos. Se utiliza junto con la etiqueta <figcaption> para agregar una leyenda o descripción del contenido multimedia, por ejemplo:

HTML 
<figure>
  <img src="imagen.jpg" alt="Descripción de la imagen" />
  <figcaption>Leyenda de la imagen</figcaption>
</figure>

<div> </div>

Es el contenedor genérico de HTML. No tiene significado semántico, por lo que se recomienda usarlo solo cuando ninguna etiqueta semántica como <section>, <article> o <aside> sea adecuada. Es útil para agrupar elementos con fines de estilo o layout, por ejemplo:

HTML 
<div class="tarjeta">
  <img src="foto.jpg" alt="Foto" />
  <p>Descripción de la tarjeta</p>
</div>

<details> </details>

Crea un elemento interactivo que el usuario puede abrir y cerrar sin necesidad de JavaScript. Se utiliza junto con la etiqueta <summary> que define el título visible del elemento. El atributo open permite que el contenido se muestre abierto por defecto, por ejemplo:

HTML 
<details>
  <summary>¿Qué es HTML?</summary>
  <p>HTML es el lenguaje de marcado estándar para crear páginas web.</p>
</details>

<details open>
  <summary>Esta sección está abierta por defecto</summary>
  <p>El atributo open hace que el contenido sea visible al cargar.</p>
</details>
lightbulb

Tip

Usar etiquetas semánticas como <header>, <main> y <footer> en lugar de <div> mejora la accesibilidad, el SEO y la legibilidad del código.
