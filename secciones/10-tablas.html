Etiquetas para tablas

Las tablas en HTML permiten organizar y presentar datos de forma estructurada en filas y columnas. Son esenciales para mostrar información tabular de manera clara y accesible.

<table> </table>

Define la estructura principal de una tabla. Dentro de ella se utilizan filas <tr>, celdas de encabezado <th> y celdas de datos <td>, por ejemplo:

HTML 
<table>
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
    <th>País</th>
  </tr>
  <tr>
    <td>Ana</td>
    <td>25</td>
    <td>Argentina</td>
  </tr>
  <tr>
    <td>Carlos</td>
    <td>30</td>
    <td>México</td>
  </tr>
</table>
Resultado
Nombre Edad País
Ana 25 Argentina
Carlos 30 México

<tr> </tr>

Define una fila (row) dentro de una tabla. Cada fila contiene celdas que pueden ser de encabezado <th> o de datos <td>. Es el contenedor horizontal que estructura la información en columnas, por ejemplo:

HTML 
<table>
  <tr>
    <th>Producto</th>
    <th>Stock</th>
  </tr>
  <tr>
    <td>Teclado</td>
    <td>45</td>
  </tr>
  <tr>
    <td>Mouse</td>
    <td>68</td>
  </tr>
</table>
Resultado
Producto Stock
Teclado 45
Mouse 68

<th> </th>

Define una celda de encabezado (header) en una tabla. Por defecto, el texto se muestra en negrita y centrado. Es importante usar el atributo scope para mejorar la accesibilidad, indicando si el encabezado aplica a una columna scope="col" o a una fila scope="row", por ejemplo:

HTML 
<table>
  <thead>
    <tr>
      <th scope="col">Mes</th>
      <th scope="col">Ingresos</th>
      <th scope="col">Gastos</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">Enero</th>
      <td>$50,000</td>
      <td>$30,000</td>
    </tr>
    <tr>
      <th scope="row">Febrero</th>
      <td>$55,000</td>
      <td>$32,000</td>
    </tr>
  </tbody>
</table>
Resultado
Mes Ingresos Gastos
Enero $50,000 $30,000
Febrero $55,000 $32,000

<td> </td>

Define una celda de datos (data) estándar en una tabla. Contiene el contenido regular de la tabla. Por defecto, el texto se alinea a la izquierda y no está en negrita. Puede contener cualquier tipo de contenido HTML como texto, imágenes, enlaces, etc., por ejemplo:

HTML 
<table>
  <tr>
    <th>Usuario</th>
    <th>Email</th>
    <th>Estado</th>
  </tr>
  <tr>
    <td>María García</td>
    <td>maria@email.com</td>
    <td>Activo</td>
  </tr>
  <tr>
    <td>Juan Pérez</td>
    <td>juan@email.com</td>
    <td>Inactivo</td>
  </tr>
</table>
Resultado
Usuario Email Estado
María García maria@email.com Activo
Juan Pérez juan@email.com Inactivo

<thead>, <tbody>, <tfoot>

Permiten organizar la tabla en tres secciones semánticas: encabezado, cuerpo y pie. Esto mejora la legibilidad, accesibilidad y permite que los navegadores apliquen estilos específicos a cada sección. El <tfoot> se usa típicamente para totales o resúmenes, por ejemplo:

HTML 
<table>
  <thead>
    <tr>
      <th>Producto</th>
      <th>Cantidad</th>
      <th>Precio</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Laptop</td>
      <td>2</td>
      <td>$2,400</td>
    </tr>
    <tr>
      <td>Mouse</td>
      <td>5</td>
      <td>$100</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="2">Total</td>
      <td>$2,500</td>
    </tr>
  </tfoot>
</table>
Resultado
Producto Cantidad Precio
Laptop 2 $2,400
Mouse 5 $100
Total $2,500

<caption> </caption>

Proporciona un título o descripción para la tabla. Debe ser el primer elemento hijo de <table>. Mejora la accesibilidad al darle contexto a los lectores de pantalla, por ejemplo:

HTML 
<table>
  <caption>Ventas del primer trimestre 2025</caption>
  <thead>
    <tr>
      <th>Mes</th>
      <th>Ventas</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Enero</td>
      <td>$15,000</td>
    </tr>
    <tr>
      <td>Febrero</td>
      <td>$18,500</td>
    </tr>
  </tbody>
</table>
Resultado
Ventas del primer trimestre 2025
Mes Ventas
Enero $15,000
Febrero $18,500

Atributos importantes

Las tablas soportan atributos especiales que permiten fusionar celdas y mejorar la estructura:

colspan

Permite que una celda se extienda sobre múltiples columnas. El valor indica cuántas columnas debe ocupar. Es útil para crear encabezados que abarquen varias columnas, por ejemplo:

HTML 
<table>
  <tr>
    <th colspan="3">Información del estudiante</th>
  </tr>
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
    <th>Calificación</th>
  </tr>
  <tr>
    <td>María</td>
    <td>22</td>
    <td>9.5</td>
  </tr>
</table>
Resultado
Información del estudiante
Nombre Edad Calificación
María 22 9.5

rowspan

Permite que una celda se extienda sobre múltiples filas. El valor indica cuántas filas debe ocupar. Es útil cuando se necesita agrupar información verticalmente, por ejemplo:

HTML 
<table>
  <tr>
    <th>Departamento</th>
    <th>Empleado</th>
    <th>Cargo</th>
  </tr>
  <tr>
    <td rowspan="2">Ventas</td>
    <td>Juan Pérez</td>
    <td>Gerente</td>
  </tr>
  <tr>
    <td>Ana García</td>
    <td>Vendedora</td>
  </tr>
  <tr>
    <td rowspan="2">IT</td>
    <td>Carlos López</td>
    <td>Desarrollador</td>
  </tr>
  <tr>
    <td>Laura Martín</td>
    <td>DBA</td>
  </tr>
</table>
Resultado
Departamento Empleado Cargo
Ventas Juan Pérez Gerente
Ana García Vendedora
IT Carlos López Desarrollador
Laura Martín DBA
lightbulb

Tip

Usar <thead>, <tbody> y <tfoot> mejora la accesibilidad y permite a los navegadores imprimir encabezados de tabla en cada página cuando una tabla abarca múltiples páginas. Evita usar el atributo border y prefiere CSS para estilizar las tablas.
