etiquetas para tablas

Las etiquetas más comunes para crear tablas en HTML son:

  • <table> </table>: Define la estructura principal de una tabla.

    Dentro de ella se utilizan filas (<tr>) y celdas (<th>).

    HTML
    <table border="1">
      <tr>
        <th>Nombre</th>
        <th>Edad</th>
        <th>País</th>
      </tr>
      <tr>
        <td>Ana</td>
        <td>25</td>
        <td>Argentina</td>
      </tr>
    </table>
    Resultado
    Nombre Edad País
    Ana 25 Argentina

  • <thead>, <tbody> y <tfoot>: Permiten organizar la tabla en encabezado, cuerpo y pie.

    Mejoran la legibilidad y accesibilidad de las tablas.

    HTML
    <table border="1">
      <thead>
        <tr>
          <th>Producto</th>
          <th>Precio</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td>Laptop</td>
          <td>$1200</td>
        </tr>
      </tbody>
      <tfoot>
        <tr>
          <td>Total</td>
          <td>$1200</td>
        </tr>
      </tfoot>
    </table>
    Resultado
    Producto Precio
    Laptop $1200
    Total $1200
