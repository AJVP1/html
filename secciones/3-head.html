Etiquetas dentro del head

Algunas de las etiquetas más comunes dentro del head son:

<meta />

Charset

Define la semántica del navegador mediante el atributo "charset", y permite el uso de caracteres especiales mediante el valor "UTF-8", por ejemplo:

HTML 
<meta charset="UTF-8" />

Author

Define el autor de la página web colocando el valor "author" en el atributo name, y el nombre del autor en el atributo content, por ejemplo:

HTML 
<meta name="author" content="Juan Pérez" />

Description

Define el uso de la página web colocando el valor "description" en el atributo name, y la descripción en el atributo content, por ejemplo:

HTML 
<meta name="description" content="Página web sobre HTML" />

Viewport

Permite el responsive design colocando el valor "viewport" en el atributo name, y el valor "width=device-width, initial-scale=1.0" en el atributo content, por ejemplo:

HTML 
<meta name="viewport" content="width=device-width, initial-scale=1.0" />

Keywords

Define las palabras clave de la página web para motores de búsqueda colocando el valor "keywords" en el atributo name, y las palabras clave separadas por comas en el atributo content, por ejemplo:

HTML 
<meta name="keywords" content="HTML, etiquetas, meta" />

Robots

Controla cómo los motores de búsqueda indexan y rastrean la página. Los valores más comunes son "index" (permitir indexación), "noindex" (no indexar), "follow" (seguir enlaces) y "nofollow" (no seguir enlaces), por ejemplo:

HTML 
<meta name="robots" content="index, follow" />

Theme-color

Define el color de la barra de dirección del navegador en dispositivos móviles, por ejemplo:

HTML 
<meta name="theme-color" content="#141414" />

Open Graph

Las meta etiquetas Open Graph controlan cómo se muestra la página cuando se comparte en redes sociales como Facebook, Twitter o WhatsApp. Se usan los atributos "property" y "content", por ejemplo:

HTML 
<meta property="og:title" content="Mi Página Web" />
<meta property="og:description" content="Descripción de la página" />
<meta property="og:image" content="https://ejemplo.com/imagen.png" />

<title> </title>

Define el nombre de la ventana en el navegador colocando el valor entre las etiquetas de apertura y cierre, por ejemplo:

HTML 
<title> Página sobre HTML </title>

<script> </script>

Define cual será el archivo de JavaScript que vamos a usar sobre el archivo HTML, se coloca la ruta del archivo JS dentro del atributo src y se usa el atributo defer para que el JavaScript se cargue después de que se haya cargado todo el archivo HTML y CSS, así se evitan errores cuando el archivo JS intenta acceder a elementos HTML que todavía no existen en la página, por ejemplo:

HTML 
<script src="ruta/del/archivo.js" defer> </script>

<link />

Stylesheet

Permite importar el diseño CSS colocando el valor "stylesheet" en el atributo rel, y la ruta del archivo CSS en el atributo href, por ejemplo:

HTML 
<link rel="stylesheet" href="ruta/del/archivo.css" />

Icon

Permite agregar un icono al lado del title colocando "icon" en el atributo rel y la ruta del icono en el atributo href, por ejemplo:

HTML 
<link rel="icon" href="ruta/del/icono.ico" />

Canonical

Indica a los motores de búsqueda cuál es la URL principal de una página cuando existen varias URLs con el mismo contenido, evitando problemas de contenido duplicado en SEO, por ejemplo:

HTML 
<link rel="canonical" href="https://ejemplo.com/pagina" />

Preconnect y Preload

Optimizan el rendimiento de la página. "preconnect" establece una conexión anticipada con un servidor externo, y "preload" carga recursos importantes de forma anticipada antes de que el navegador los necesite, por ejemplo:

HTML 
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preload" href="estilos.css" as="style" />

<style> </style>

Permite escribir estilos CSS directamente dentro del documento HTML sin necesidad de un archivo externo. Es útil para estilos pequeños o específicos de una sola página, por ejemplo:

HTML 
<style>
  body {
    font-family: Arial, sans-serif;
    background-color: #f7f7f7;
  }
</style>

<base />

Define la URL base para todas las URLs relativas del documento. También puede definir cómo se abren los enlaces por defecto mediante el atributo "target", por ejemplo:

HTML 
<base href="https://ejemplo.com/" target="_blank" />

<noscript> </noscript>

Define contenido alternativo que se muestra cuando el navegador del usuario no soporta JavaScript o lo tiene deshabilitado, por ejemplo:

HTML 
<noscript>
  <p>Esta página requiere JavaScript para funcionar correctamente.</p>
</noscript>
lightbulb

Tip

Las etiquetas <meta>, <link> y <script> no son visibles en la página, pero son fundamentales para el SEO, el rendimiento y el correcto funcionamiento del sitio.
