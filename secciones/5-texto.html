Etiquetas de tipo texto

Estas etiquetas se utilizan para dar formato y estructura al texto dentro de la página web.

<h1> a <h6>

Define los encabezados de diferentes niveles, siendo <h1> el más importante y <h6> el menos importante, por ejemplo:

HTML 
<h1>Encabezado nivel 1</h1>
<h2>Encabezado nivel 2</h2>
<h3>Encabezado nivel 3</h3>
<h4>Encabezado nivel 4</h4>
<h5>Encabezado nivel 5</h5>
<h6>Encabezado nivel 6</h6>

Resultado

Encabezado nivel 1

Encabezado nivel 2

Encabezado nivel 3

Encabezado nivel 4

Encabezado nivel 5
Encabezado nivel 6

<p> </p>

Define un párrafo de texto. El navegador agrega automáticamente un espacio antes y después de cada párrafo, por ejemplo:

HTML 
<p>Este es un párrafo de texto.</p>
<p>Este es otro párrafo de texto.</p>

<strong> </strong>

Define texto con énfasis fuerte. El navegador lo muestra en negrita, por ejemplo:

HTML 
<p>Este texto es <strong>muy importante</strong>.</p>

Resultado

Este texto es muy importante.

<em> </em>

Define texto con énfasis moderado. El navegador lo muestra en cursiva, por ejemplo:

HTML 
<p>Este texto tiene <em>énfasis</em>.</p>

Resultado

Este texto tiene énfasis.

<span> </span>

Es un contenedor en línea sin significado semántico. Permite identificar una pequeña parte del texto para cambiarle el estilo, por ejemplo:

HTML 
<p>El precio es <span style="color: red;">$99.99</span></p>

<br />

Inserta un salto de línea dentro de un párrafo, por ejemplo:

HTML 
<p>Primera línea<br />Segunda línea</p>

<hr />

Inserta una línea horizontal que sirve como separador de contenido, por ejemplo:

HTML 
<p>Sección uno</p>
<hr />
<p>Sección dos</p>

Resultado

Sección uno

Sección dos

<blockquote> </blockquote>

Define una cita en bloque, generalmente se utiliza para citas largas o extractos de texto. El atributo cite permite indicar la fuente de la cita. Se puede usar junto con la etiqueta <cite> para mostrar la referencia visible, por ejemplo:

HTML 
<blockquote cite="https://ejemplo.com">
  Esta es una cita en bloque.
</blockquote>
<cite>MDN Documentation</cite>

Resultado

Esta es una cita en bloque.
MDN Documentation

<q> </q>

Es una cita a nivel de línea y no de párrafo. El navegador la muestra entre comillas automáticamente, por ejemplo:

HTML 
<p>El autor dijo <q cite="https://ejemplo.com">Esta es una cita en línea.</q></p>

Resultado

El autor dijo Esta es una cita en línea.

<abbr> </abbr>

Para abreviaciones. El navegador muestra el texto subrayado con puntos y al pasar el cursor muestra el significado completo mediante el atributo title, por ejemplo:

HTML 
<abbr title="HyperText Markup Language">HTML</abbr>

Resultado

HTML

<address> </address>

Para direcciones o información de contacto. El navegador muestra el texto en cursiva, por ejemplo:

HTML 
<address>123 Main St, Anytown, USA</address>

Resultado

123 Main St, Anytown, USA

<sup> </sup>

Para superíndices. El navegador muestra el texto elevado y en un tamaño menor, por ejemplo:

HTML 
10<sup>2</sup>

Resultado

102

<sub> </sub>

Para subíndices. El navegador muestra el texto hundido y en un tamaño menor, por ejemplo:

HTML 
H<sub>2</sub>O

Resultado

H2O

<time> </time>

Para representar fechas y horas. Usa el atributo datetime para especificar la fecha y hora en formato ISO 8601, por ejemplo:

HTML 
<p>Hoy <time datetime="2023-03-15">15 de marzo de 2023</time> se celebra el cumpleaños de Ana.</p>

Resultado

Hoy 15 de marzo de 2023 se celebra el cumpleaños de Ana.

<a> </a>

Crea enlaces (hipervínculos). Usa el atributo href para especificar la URL del enlace. También puede usarse con protocolos especiales como mailto: o tel:, por ejemplo:

HTML 
<a href="https://www.ejemplo.com">Visitar ejemplo.com</a>

<a href="https://wa.me/5700000">Enviar WhatsApp</a>

Resultado

Visitar ejemplo.com

Abrir en nueva pestaña

Usa el atributo target="_blank" para abrir el enlace en una nueva pestaña. Si usas _blank, es necesario agregar rel="noopener noreferrer" por temas de seguridad, por ejemplo:

HTML 
<a href="https://www.ejemplo.com" target="_blank" rel="noopener noreferrer">
  Abrir en nueva pestaña
</a>
lightbulb

Tip

El uso correcto de etiquetas de texto es muy importante para el SEO. Los motores de búsqueda usan etiquetas como <h1>, <strong> y <a> para entender la estructura y relevancia del contenido.
